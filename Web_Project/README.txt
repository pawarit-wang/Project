เว็ปไซต์นี้เป็นเว็ปไซต์ขายเกม 

ขั้นตอนแรกให้เปิดไฟล์ที่ชื่อว่า main.html ขึ้นมา จะเป็นการเริ่มเข้าหน้าเว็ปไซต์หลัก โดยจะมีแถบด้านบนไว้ใช้สำหรับการกดเพื่อไปหน้าอื่นๆ 
และรูปเกมต่างๆ ที่สามารถกดเพื่อเข้าไปดูรายลละเอียดเพิ่มเติมได้ 

หากกดไปหน้า search จะมีหมวดหมู่ด้านซ้ายที่กดเพื่อเลือกหมวดหมู่ได้

about us เป็นหน้าสำหรับการแสดงสมาชิกที่สร้างเว็ปไซต์นี้

และ login เป็นปุ่มเข้าสู่ระบบสำหรับผู้ดูแลระบบ โดยการเข้าสู่ระบบจะประกอบด้วยการใส่ email และ password
หากเข้าสู่ระบบเข้าไปจะเป็นหน้าเว็ปสำหรับผู้ดูแลระบบ ซึ่งจะมีปุ่ม edit item และ edit admin เพิ่มมาทางด้านแถบด้านบน

โดยหากกด edit item จะเป็นหน้าสำหรับการ เพิ่ม, แก้ไข หรือลบสินค้า

และปุ่ม edit admin จะเป็นหน้าสำหรับการ เพิ่ม, แก้ไข หรือลบข้อมูลของผู้ดูแลระบบ

หากต้องการที่จะออกจากระบบให้กดปุ่ม logout จะเป็นการกลับไปยังหน้า main
